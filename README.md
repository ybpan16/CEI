# CEI

Tools for joint detection of expanded TCR clones using Mahalanobis distances with permutation and ACAT, under FDR control.

## Installation

```bash
pip install git+https://github.com/ybpan16/cei.git


## Run-example
cei -i ./tests/CLE0083_BP1.xlsx --Bcol "B" --Pcol "P1" --out-joint ./tests/CLE0083_BP1_joint.xlsx
